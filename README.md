Objetivo
Crear una app móvil gamificada que convierta la vida en un videojuego; combina seguimiento de hábitos, rutinas guiadas estilo Fabulous y microjuegos domésticos tipo Oracle of Dishes, con estética mística y automatización con IA.
MVP resumido
- Registro y perfil con avatar.
- Sistema de hábitos con streaks y XP.
- Rutinas guiadas (1 rutina inicial: mañana o noche) con pasos y timers.
- Microjuego Oracle para tareas domésticas (temporizador 30–90s, resultado aleatorio con bonus).
- Misiones diarias/semanales y logros.
- Tienda cosmética y moneda ligada a progresión.
- Dashboard con rutina del día, racha y botón rápido “Jugar Oracle”.
- Mock API local para que frontend avance.
